# **App Name**: Wonosobo Explorer

## Core Features:

- Landing Page with Hero Section: Full-screen hero section on index.html with a background image of Sikunir and the text "Eksplorasi Wonosobo dengan Lokal Guide Terbaik".
- Service Packages Grid: Display Our Services in a 3-column grid, detailing the Sunrise Package (Penjemputan 03.00, Guide, Dokumentasi), Reguler Package (Candi Arjuna, Kawah Sikidang, Telaga Warna), and Car Rental (Avanza, Innova, Hiace) with Driver = Guide Lokal note.
- WhatsApp Booking Button: Implement a static WhatsApp button linking to https://wa.me/628xxx?text=Halo%20saya%20mau%20pesan%20paket%20[Nama] for direct booking.
- Blog Listing Page: Create a page (artikel.html) displaying a collection of article cards with attractive thumbnails, covering topics like "Kuliner Mie Ongklok", "Ritual Rambut Gimbal", and "Tips Cuaca Dieng".
- Article Detail Template: Design a clean article layout (detail-artikel.html) focusing on text and images, with a prominent "Kembali ke Beranda" button.
- SEO Optimization: Include relevant <title>, <meta description>, and OpenGraph tags for improved SEO and link previews.
- Image Placeholders: Use image placeholders with 16:9 aspect ratio to prevent layout shifts during loading.

## Style Guidelines:

- Primary color: Pine Green (#1A3922) to evoke a sense of nature and tranquility. This dark color is suitable for the light color scheme.
- Background color: Mist White (#F8F9FA) to provide a clean and airy feel, complementing the Pine Green.
- Accent color: Sun Gold (#D4AF37) for buttons and highlights, drawing attention and providing a warm contrast. The accent color, sun gold, is about 30 degrees to the "left" of the Pine Green, on the color wheel.
- Body and headline font: 'PT Sans' for a modern and slightly warm feel.
- Mobile-first design with a hamburger menu on smaller screens for better navigation.
- Subtle hover effects on package cards (slight lift/shadow) for added interactivity.
- Smooth scrolling effect for seamless transitions between sections.
